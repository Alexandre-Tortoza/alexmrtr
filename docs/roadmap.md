# Roadmap

```
🟢 Phase 1 — Setup (highest priority)
  ├── Tailwind + globals.css
  ├── Vitest + test setup
  ├── i18n config + initial dictionary
  └── Netlify config

🟡 Phase 2 — Shared UI primitives
  ├── Button, Card, Container, Section, SEO
  ├── component tests
  └── ThemeToggle

🟡 Phase 3 — Entities + Data layer
  ├── types: Project, Post, Photo
  ├── validation schemas (Zod)
  └── sample .mdx content

🟠 Phase 4 — Features
  ├── blog (PostCard, PostList, api)
  ├── projects (ProjectCard, ProjectGrid, api)
  ├── photos (PhotoGrid, Lightbox, api)
  └── i18n (LanguageSwitcher, LocaleText)

🔵 Phase 5 — Pages
  ├── Landing (index.astro)
  ├── Projects
  ├── Blog (list + [slug])
  └── Photos

🟣 Phase 6 — Content + Tests
  ├── Real blog posts
  ├── Real projects
  ├── Photos
  └── Integration tests

⚪ Phase 7 — Performance + SEO
  ├── Lighthouse audit
  ├── Open Graph tags
  ├── Sitemap
  └── RSS Feed
```
