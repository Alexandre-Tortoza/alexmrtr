# Internationalization (PT-BR + EN)

- **No external lib** — local dictionary in `shared/lib/i18n.ts`
- Nested keys: `blog.title.pt`, `blog.title.en`
- `<T key="blog.title" />` component renders text for the active locale
- Locale detection order:
  1. Query param `?lang=en`
  2. Cookie `locale`
  3. Browser `Accept-Language`
  4. Fallback: `pt`
- URL structure: `site.com/blog` (no locale prefix — negotiates via `Accept-Language` + cookie)
