# Alexandre Tortoza

Um portfolio digital criado para unir tecnologia, identidade visual e experiencias interativas.

Mais do que uma pagina pessoal, este projeto funciona como uma vitrine criativa para apresentar projetos, ideias, experimentos visuais e conteudos sobre desenvolvimento web. Cada detalhe foi pensado para transmitir personalidade, performance e cuidado com a experiencia de quem navega.

## Sobre o Projeto

Este portfolio nasceu como um espaco para conectar design, codigo e narrativa. A proposta e simples: apresentar quem eu sou, o que eu construo e como penso tecnologia de uma forma visualmente marcante, leve e direta.

Aqui voce encontra uma combinacao de paginas institucionais, blog, projetos, galeria e interacoes visuais que tornam a navegacao mais envolvente.

## O Que Voce Encontra Aqui

- Uma interface moderna, responsiva e focada em performance
- Animacoes suaves que tornam a navegacao mais fluida
- Experimentos visuais com elementos interativos e efeitos 3D
- Blog para compartilhar ideias, aprendizados e experiencias
- Projetos selecionados com foco em tecnologia, criatividade e resultado
- Uma identidade visual propria, pensada para se destacar sem perder clareza

## Tecnologias em Destaque

Este projeto foi construido com uma stack moderna voltada para velocidade, interatividade e qualidade visual.

**Astro** entrega uma base leve e extremamente performatica, ideal para criar uma experiencia rapida desde o primeiro carregamento.

**Tailwind CSS** ajuda a transformar ideias visuais em interfaces consistentes, responsivas e com acabamento moderno.

**Three.js** adiciona uma camada mais imersiva ao portfolio, permitindo criar elementos visuais interativos e experiencias inspiradas em WebGL.

**GSAP** traz animacoes mais naturais, precisas e profissionais, elevando a sensacao de fluidez durante a navegacao.

**TypeScript** reforca a qualidade do codigo e permite construir uma base mais confiavel, organizada e preparada para evoluir.

**Markdown** torna o conteudo mais flexivel, facilitando a publicacao de posts, ideias e materiais dentro do proprio portfolio.

## A Ideia por Tras

O objetivo deste portfolio e mostrar que tecnologia tambem pode comunicar personalidade. Performance, acessibilidade visual, movimento e conteudo caminham juntos para criar uma presenca digital que nao seja apenas funcional, mas memoravel.
